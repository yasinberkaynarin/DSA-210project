# DSA-210project
Physical Intensity vs Technical Quality in the Premier League (2020–2024)
Project Goal

This project investigates how physical metrics and technical metrics evolved across Premier League seasons from 2020–21 to 2023–24, and whether these changes are associated with team success.

Data Sources
-FBref data (via soccerdata Python library)
-StatsBomb Open Data
Current Milestone
-Data collection
-Data cleaning
-Exploratory Data Analysis
-Hypothesis testing
