# DSA-210project
Spotify Popularity Analysis
Project Overview

This project analyzes which factors influence song popularity on Spotify.

The dataset includes song-level audio features such as danceability, energy, loudness, tempo, and artist level enrichment variables such as followers and artist popularity.

Research Question

Which factors have a stronger effect on song popularity: audio features or artist fame?

Hypothesis

Danceability has a stronger effect on popularity than artist followers.

Methods
Data Cleaning
Exploratory Data Analysis (EDA)
Correlation Analysis
Hypothesis Testing
Data Visualization
Results

The hypothesis was rejected.

Danceability showed almost no relationship with popularity, while followers had a stronger positive relationship.
